# Analyse de données médicales – Cancer du sein

## Contexte
Analyse exploratoire de données médicales pour comprendre
les différences biologiques entre tumeurs bénignes et malignes.

## Données
Dataset Breast Cancer Wisconsin contenant des mesures cellulaires.

## Méthodes
- Python
- Pandas
- Matplotlib
- Analyse descriptive

## Résultats
Les tumeurs malignes présentent en moyenne des valeurs plus élevées
pour certaines caractéristiques cellulaires.

## Intérêt
Projet utile pour la recherche et l’aide à la décision en santé publique.
